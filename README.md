NESTJS ADMIN API

## Description

Control of User, Role and Companies, CRUD, Authentication, Test with memory sqlite3 in memory



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:development

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
# Specific Test
$ npx jest --testPathPattern=src/role/role.controller.spec.ts

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
